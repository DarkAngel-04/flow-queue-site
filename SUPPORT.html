Flow Queue Support

Flow Queue is supported by Abhishek Kumar. For help, bug reports, or privacy questions, email abhishekrajeevnagar@gmail.com.

Before contacting support

  1. Confirm that a Google Flow project is open in the active tab.
  2. Reload Flow Queue from chrome://extensions.
  3. Retry with Concurrent set to 1.
  4. Open Diagnostics, export the logs, and attach them only after checking that they contain no information you do not want to share.

Include in a bug report

  • Chrome version
  • Flow Queue version
  • Selected generation mode
  • Steps to reproduce the issue
  • A screenshot with sensitive information removed
