Flow Queue Privacy Policy

Effective date: June 20, 2026

Flow Queue is a Chrome extension that automates user-directed image and video prompt queues in Google Flow.

Data Flow Queue handles

  • Prompts entered or imported by the user
  • Reference images selected by the user
  • Google Flow page content needed to locate controls and monitor generation status
  • URLs and filenames of generated media selected for download
  • Extension settings, language, appearance, run status, and diagnostic logs
  • Cookies scoped to labs.google, only when the user explicitly selects Clear cache

Use and storage

This data is used only to run user-requested queues, monitor generation, organize optional downloads, save local preferences, show local diagnostics, and clear the Google Flow session when explicitly requested. Settings and logs are stored locally using Chrome extension storage. Flow Queue does not use Chrome Sync.

Sharing and third parties

Flow Queue does not operate a developer server and does not transmit user data to Abhishek Kumar, advertising networks, analytics providers, data brokers, or other third parties.

When the user runs a queue, selected prompts and reference images are submitted to Google Flow at the user's direction. Google processes that content under Google's own terms and privacy policy. Downloaded media is handled by Chrome's download system.

Cookies and clipboard

Flow Queue does not use cookies for tracking. Clear cache accesses labs.google cookies only to delete them. Cookie values are not stored, logged, or transmitted. The extension may write user-selected content or diagnostic text to the clipboard, but it does not read clipboard contents.

Data sales and advertising

Flow Queue does not sell user data and does not use it for advertising, profiling, creditworthiness, lending, or unrelated purposes.

Security and deletion

Flow Queue limits access to labs.google, uses extension-packaged executable code, and communicates with Google Flow over HTTPS. Users can delete diagnostic logs inside the extension and remove all locally stored extension data by uninstalling it.

Chrome Web Store Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
